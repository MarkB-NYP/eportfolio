Project Scope

During the September holidays in 2020, I was looking for a way to practice the web design and Javascript coding skills I learnt during my first semester at Nanyang Polytechnic.

Snipptet Writer (originally called Smiley Writer) is a fun minimalist text editor that only allows you to write 50 words. It's for drafting short messages or making notes.

My Role And Solution

This was a personal project, so I did all design and coding myself.

For faster loading speeds and extra security, Snippet Writer runs entirely in the user's browser. Instead of saving of saving output to a text file or in the cloud, I chose to allow the user to copy text to their device's clipboard so they can paste it into the messaging app of their choice.

This is especially useful on mobile devices, and was also much easier to implement.

My Work Process

I started by drawing a wireframe of the app. Then, I created a few different prototypes with slightly different fonts, layouts and features. I used them all for a week, then picked the version that I felt was the most comfortable to use.

At the time, I was a beginner to Javascript programming. It took me a few afternoons to get all the code and animations right, but I had fun!

Outcome and Results

For a long time, I used Snippet Writer as an offline HTML file. I shared a few copies of the code with my family and received positive feedback.

Recently I discovering GitHub Pages, and I decided to host the code online that anyone can use Snippet Writer anywhere.
I hope to keep working on the project and share it with even more people in the future.
